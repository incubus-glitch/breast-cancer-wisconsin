# 🧠 Breast Cancer Classification (Machine Learning Project)

Проект посвящён задаче бинарной классификации опухолей молочной железы на основе набора числовых признаков, описывающих их геометрические характеристики.

Цель работы — сравнить различные модели машинного обучения и определить наиболее эффективный подход для данного датасета.

---

## 📊 Основные этапы работы

В рамках проекта были выполнены следующие шаги:

- проведён исследовательский анализ данных (EDA)
- проверена корреляция признаков и их распределения
- выполнено понижение размерности с помощью PCA
- протестированы различные модели машинного обучения
- проведена кросс-валидация
- выполнена оптимизация гиперпараметров (GridSearchCV)

---

## 🤖 Использованные модели

В работе были рассмотрены следующие алгоритмы:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- XGBoost

---

## 📈 Результаты

### 📌 Logistic Regression
- Accuracy: ~0.935
- F1-score: ~0.910

### 📌 KNN
- Accuracy: ~0.933
- F1-score: ~0.908

### 📌 SVM
- Mean F1-score: ~0.903
- После оптимизации (GridSearchCV):
  - Accuracy: 0.982
  - F1-score: 0.976

### 📌 XGBoost
- Mean F1-score: ~0.898

---

## 🏁 Вывод

На данном датасете наблюдается высокая линейная разделимость классов, благодаря чему:

- простые модели (Logistic Regression) показывают стабильные и сильные результаты
- сложные модели (SVM, XGBoost) без настройки не дают значимого прироста качества
- после подбора гиперпараметров SVM достигает наилучшего результата среди всех моделей

В целом задача классификации эффективно решается классическими методами машинного обучения, а ключевым фактором качества является не сложность модели, а информативность исходных признаков.
